# Noviq Now

Real-time news tracking dashboard with AI-powered summarization and real-time updates from multiple sources.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3003](http://localhost:3003) with your browser to see the result.

## Features

- **Real-time news feed** - Live updates from multiple news sources
- **AI-powered summarization** - Get concise summaries of articles
- **Category filtering** - Filter by Technology, AI, Trading, Wars, etc.
- **Bookmarks** - Save articles for later reading
- **Notifications** - Get notified of breaking news
- **Dark/Light theme** - Toggle between themes
- **Responsive design** - Works on desktop and mobile

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Framer Motion
